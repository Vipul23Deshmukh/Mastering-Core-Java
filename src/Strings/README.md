# Strings

Everything related to String class and string handling in Java.

## What I practiced

- String creation (literal vs new)
- Immutability concept
- Common methods: length(), charAt(), substring(), indexOf(), contains(), equals(), equalsIgnoreCase()
- String comparison
- Reversing a string
- Palindrome check
- StringBuilder / StringBuffer basics (later)
