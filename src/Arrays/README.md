# Arrays

Basic to intermediate array exercises and patterns.

## What I practiced here

- Array declaration & initialization
- Finding sum, max, min elements
- Searching & sorting basics
- Multi-dimensional arrays (later)

## Files

| File                      | Description                              |
|---------------------------|------------------------------------------|
| Arrays1.java              | Basic array creation and printing        |
| Largest_Element.java      | Find maximum element in array            |
| Sum_of_Array.java         | Calculate sum of all elements            |

Most examples use simple loops – no library methods yet.
